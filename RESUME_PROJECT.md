# Resume Project

## 中文版

项目名称：NexusAgent 企业级 AI 客服与知识库智能体平台

- 使用 FastAPI、React、PostgreSQL 与 pgvector 构建完整 AI 应用，覆盖后端 API、前端交互、数据库持久化和 Docker 部署准备。
- 实现 RAG 检索、Citation 溯源、Hybrid Intent Routing 和 Tool Calling，支持订单查询、库存查询、工单创建和人工转接等可审计流程。
- 建立 Evaluation、Request Metrics、Docker 配置和 GitHub Actions CI Pipeline，提升项目可测试性、可展示性和作品集交付质量。

## English Version

Project: NexusAgent - AI Support Agent with PostgreSQL pgvector RAG

- Built a full-stack AI support application with FastAPI, React, PostgreSQL, pgvector, Async SQLAlchemy repositories, and Docker deployment preparation.
- Implemented RAG retrieval, source citations, hybrid intent routing, and typed business tool calling for order lookup, inventory lookup, support tickets, and human handoff.
- Added deterministic evaluation, request metrics, smoke testing, and GitHub Actions CI to make the project testable and portfolio-ready.
