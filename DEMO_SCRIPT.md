# Demo Script

## 60-90 Second Recording Plan

0-10s: Show GitHub repo or landing page.

Say: `NexusAgent is a full-stack AI customer support and knowledge agent built with FastAPI, React, PostgreSQL pgvector, RAG citations, and typed business tools.`

10-25s: Ask `What is NovaTech's return policy?`

Show RAG answer, citation, and `return_policy.md` source.

25-40s: Ask `Where is order ORD-10001?`

Show detected order intent, `get_order_status` tool execution, and order result.

40-55s: Ask `My keyboard stopped working. Please create a support ticket.`

Show ticket creation and the `TCK-...` public ID.

55-70s: Ask `Do you offer drone insurance?`

Show no-context refusal instead of a fabricated answer.

70-90s: Show Knowledge Base, Analytics, architecture diagram, GitHub repo, and live demo URL if deployed.

Close with: `The project includes deterministic evaluation, CI, Docker deployment planning, and a smoke test script for release verification.`
