# Demo Script

1. Start the stack with `docker compose up --build` or run backend/frontend locally.
2. Open the frontend and ask: `What is NovaTech's return policy?`
3. Show that the answer includes a citation from `return_policy.md`.
4. Ask: `Where is order ORD-10001?`
5. Show the tool execution and order status from the database-backed repository.
6. Ask: `My keyboard stopped working. Please create a support ticket.`
7. Open Tickets and show the generated `TCK-...` public ID.
8. Ask: `Do you offer drone insurance?`
9. Show the no-context refusal instead of a fabricated answer.
10. Mention that evaluation output is available in `backend/evals/EVAL_REPORT.md`.
